# Cats and Dogs Classification using SVM

## Objective
Implement a Support Vector Machine (SVM) to classify images of cats and dogs.

## Tools Used
- Python
- OpenCV
- NumPy
- Scikit-learn

## Dataset
Kaggle Dogs vs Cats Dataset

## Methodology
1. Loaded cat and dog images.
2. Resized images to 64x64 pixels.
3. Converted images into numerical feature vectors.
4. Split data into training and testing sets.
5. Trained a linear SVM classifier.
6. Evaluated model performance using accuracy.

## Result
Accuracy Achieved: 54.75%

## Outcome
Successfully implemented an SVM-based image classifier for distinguishing cats and dogs.
